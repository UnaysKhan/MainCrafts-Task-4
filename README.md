# 🚀 Task Manager Dashboard (Task 4)

## 🌟 Overview

This project is a **Task Manager Dashboard** built as part of the **Full Stack Web Development Internship – Task 4**.

It is a fully interactive **frontend CRUD application** that allows users to manage tasks efficiently using **LocalStorage**.

---

## ✨ Features

### 🧩 Core Functionality

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* 🔍 Search tasks in real-time

### 💾 Storage

* Uses **LocalStorage** to persist tasks even after page refresh

### 🎨 UI/UX

* Modern **glassmorphism dashboard design**
* Smooth hover animations
* Clean task cards layout
* Responsive and user-friendly interface

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Modern UI + Glass Effect)**
* **JavaScript (CRUD + DOM + LocalStorage)**

---

## 📂 Project Structure

```bash
task-manager/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

1. User enters a task and clicks **Add Task**
2. Task is stored in **LocalStorage**
3. Tasks are dynamically displayed on the dashboard
4. User can:

   * Edit ✏️
   * Delete ❌
   * Mark complete ✔️
5. Search bar filters tasks instantly
